# 리액트 컴퍼넌트 Karma, Webpack, Mocha로 테스트하기

Karma, Webpack, Mocha를 이용한 React 컴퍼넌트 테스트 예제

### 사용
```
npm install
npm start
```

### 테스트하기

Karma를 한번만 실행한다.
```
npm test
```
테스트를 진행하며 개발 할 수 있도록 Karma를 실행한다.
```
npm run test_watch
```

- [Karma-Webpack](https://webpack.github.io/docs/usage-with-karma.html#webpackmiddleware)
